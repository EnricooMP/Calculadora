# Calculadora Interativa com Análise Numérica 🧮

Este projeto é uma calculadora desenvolvida em Python que vai além das quatro operações básicas. Além de calcular o resultado, o programa realiza uma análise detalhada das propriedades do número gerado.

## ✨ Funcionalidades

- **Operações Matemáticas:** Soma, Subtração, Multiplicação e Divisão.
- **Análise de Resultados:** Para cada cálculo, o programa informa se o número é:
  - Par ou Ímpar.
  - Positivo ou Negativo.
  - Inteiro ou Decimal.
- **Tratamento de Erros:** Utiliza blocos `try/except` para garantir que o programa não trave caso o usuário digite letras em vez de números.
- **Validação de Divisão:** Impede o erro de divisão por zero.
- **Interface Amigável:** Menu contínuo que permite realizar vários cálculos seguidos ou sair quando desejar.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python 3
- **Conceitos aplicados:** Funções (`def`), Estruturas de repetição (`while`), Condicionais (`if/elif/else`), Manipulação de Strings (`lower/strip`) e Tratamento de exceções.

1. Certifique-se de ter o **Python** instalado em sua máquina.
2. Salve o código em um arquivo chamado `main.py`.
3. Abra o terminal ou prompt de comando na pasta onde o arquivo foi salvo.
4. Execute o comando:
   ```bash
   python main.py
